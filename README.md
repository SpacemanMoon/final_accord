# final_accord
This is an app for easily finding repair professionals for all your essential needs.


"Для того чтобы пользоваться приложением, необходимо пройти этап регистрации. 
Если в дальнейшем хотите восстанавливать пароль, необходимо вводить настоящую почту. 
Далее вас ждет увлекательный, простой, логичный, интуитивно-понятный дизайн. На экране "Создать заказ" - необходимо выбрать услугу, которая вам нужна. Далее подберется список мастеров, которые могут ее оказать. Потом требуется описать Вашу проблему, согласовать дату, время - отправить заказ на обработку. Специалист свяжется с вами обкашляет детали и пошло поехало. Прошу приготовить наличку для оплаты работы мастера. По исполнению заказа. Необходимо в истории его завершить. Заказ отобразиться в списке завершенных, появится возможность поставить отзыв. Так же заказ можно отменить, но не позднее 1 часа до начала работ. Дальше заказ отменить будет нельзя, будете платить неустойку. Чтобы мастер не задавал при звонке тупых вопросов типа: "А где вы живете?" - на экране профиля можно указать город, адрес и ознакомиться с нашей компанией. Совсем скоро появится возможность ставить аватарку на профиль без багов, при перезапуске приложения".  

С вами была команда из одного человека: Designer from San-Francisco
